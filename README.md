## How To Use

Clone the project

```html
$ git clone https://github.com/Martimnio/Podcastr.git
```

Install the dependencies needed with Yarn

```html
yarn --
```

Start the server and the project

```html
yarn start
```

```html
yarn dev
```
Then acess your localhost!
